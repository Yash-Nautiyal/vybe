# VYbe

Instagram/TikTok-style Reels app built with Flutter and Firebase Firestore.

## Features

- Vertical full-screen video feed
- Fetch reel metadata from Firestore
- Local video caching and preload-ahead playback
- Auto-play / pause on scroll
- Overlay UI: username, likes, caption

## Quick start

```bash
flutter pub get
flutter run
```
